memoru
=======

* Memoru generate memos, and support to operate for memos.
* Each memo has unique and random index.

How to install
-------
```sh
git clone https://github.com/ayu-mushi/memoru
cd memoru
python setup.py install
```

tutorial
--------
```sh
memoru init
memoru gen
```
